Detailed Description

Класс для загрузки сцен и сохранения имени студента. Пока использует PlayerPrefs для временного хранения имени.

Member Function Documentation

◆ CheckName()

void SceneLoader.CheckName ( string name)

Проверяет введённое имя и сохраняет его в PlayerPrefs, если имя не пустое.

Parameters
nameИмя, введённое пользователем в input field.

В текущей реализации сохранение осуществляется через PlayerPrefs.SetString с ключом "StudentName". Это временное решение; в реальном проекте можно использовать более надёжное хранилище.
