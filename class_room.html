Detailed Description

Компонент комнаты, хранящий ссылки на соседние комнаты и управляющий активацией связанных с комнатой игровых объектов.

Комнаты могут быть соединены в четырёх направлениях: вверх, вниз, влево, вправо. При активации комнаты все объекты из массива objectsToManage также активируются; при деактивации — деактивируются.

Member Data Documentation

◆ objectsToManage

GameObject [] Room.objectsToManage

Массив игровых объектов, которые следует активировать/деактивировать вместе с данной комнатой.

Может включать визуальные эффекты, источники света и т.п. Объекты, на которые нет ссылки (null), пропускаются.
